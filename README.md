A note on scope
This is the leanest hub in the series, deliberately. The four dominant open-source quantum network simulators — SeQUeNCe, SimulaQron, QuNetSim, and QuISP — already live in the Quantum Computing Hub. What's left, once you exclude those and tools like NetSquid (which requires registration and isn't fully open-licensed), is a genuinely small pool of independent, verifiably open-source projects. Rather than pad the list, this directory only includes what could be confirmed as real and open source. That scarcity is itself a finding worth noting: quantum networking is the most under-tooled layer of the entire quantum stack.
