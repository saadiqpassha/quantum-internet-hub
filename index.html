The Quantum Networking & quantum internet tool directory

Entanglement distribution, quantum repeaters, and QKD post-processing — the software layer behind the "quantum internet." This is intentionally the leanest hub in the series.

A note on scope: the flagship quantum-network simulators (SeQUeNCe, SimulaQron, QuNetSim, QuISP) already live in the main Quantum Computing Hub. What's left is a genuinely small pool of independent, fully open-source tools — most of the field runs on those four, or on tools like NetSquid that require registration and aren't fully open-licensed. This directory only includes what's verifiably open source.
